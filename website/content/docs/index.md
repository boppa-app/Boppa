# Getting started

Boppa is an app that lets you turn any website into a native music player. Everything it can search,
browse, and play comes from a **media source**, a small config you add yourself.

## Add your first media source

Check out [r/BoppaApp](https://reddit.com/r/BoppaApp) to find a media source config.

Once you have a config URL, open **Settings → Media Sources → Add Source** and paste it in.
Boppa fetches the config, validates it, and adds the media source to your app.

## Adding a custom media source

If you just want to listen, that's it, you're set. If you'd rather build your own media
source instead of using one from the community, see [Media sources](/docs/media-sources)
for how a config is put together.
